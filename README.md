## 💼 Featured Projects

Here are some of my notable projects:

- **[Oscivox-Analog-Acoustic-Guitar-Tuner](https://github.com/tharinadacwa/Oscivox-Analog-Acoustic-Guitar-Tuner-EN2091-Analog-Project)** - The Analog Acoustic Guitar Tuner is a fully analog signal processing system that detects the frequency of a plucked guitar string and visually indicates whether the string is tuned correctly. Unlike digital tuners, this device performs all signal processing using analog circuits, including filtering, frequency detection, and voltage comparison.

- **[ThermoBand](https://github.com/tharinadacwa/ThermoBand-EN1190)** - ThermoBand a wearable temperature-monitoring device designed to support real-time health monitoring and affordable healthcare solutions for vulnerable communities.

- **[ADAPTIVE_GRAVITY_BASED_INFUSION_PUMP_MONITOR](https://github.com/tharinadacwa/BM2210-ADAPTIVE_GRAVITY_BASED_INFUSION_PUMP_MONITOR)** - A low-cost, IoT-enabled smart infusion monitoring system that upgrades traditional gravity-based IV infusion pumps with real-time monitoring, alarms, and remote ward-level visibility. Designed specifically for resource-limited hospital environments.

- **[Two-Way Digital Paging System Using Software Defined Radios (SDR)](https://github.com/tharinadacwa/Communication_Design_Project)** - Designed and implemented a two-way digital paging system using Software Defined Radios as part of the EN2130 Communication Design Project. The system enables reliable wireless transmission of short text messages between nodes with proper addressing and acknowledgment mechanisms.
